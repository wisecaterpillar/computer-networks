Описание классов:
•	Main.java  занимается взаимодействием с пользователем, считывая введенные команды для управления деятельностью сервера.
•	EMailAddress.java содержит класс, который описывает структуру e-mail адреса.
•	Message.java содержит  класс, который описывает структуру сообщения. 
•	SMTPServer.java содержит класс, который принимает подключения от клиентов и запускает персональные потоки для обработки команд с каждого из них.
•	ClientHandler.java содержит класс, который обрабатывает команды, поступивние от клиента и выполняет соответствующие действия.

Логика работы:
При старте приложения  в консоль выводится информация о командах сервера. При желании, пользователь (администратор сервера) может настроить домен и порт по умолчанию, введя команду «set», после старта сервера эта команда становится недоступна. 
Вся информация получаемая и отправляемая клиентом, записывается в файл-журнал в папке LOGs.В случае возникновения фатальных ошибок, когда сервер не может запуститься или продолжать работу, текст ошибки выводится на экран.
